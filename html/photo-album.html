Prerequisites

Before installing the photo album organizer you must check that your computer fulfills some basic requirements.

  1. Disk Space

    The application needs at least 100MB free disk space. Ask your system administrator for help To find the available free space on your disk, type df in a terminal and check that the partition where you want to install the program has at least 100 MB free space. You can see the free space in the Avail column.

    Filesystem            Size  Used Avail Use% Mounted on
/dev/md0               73G   65G  4.3G  94% /
/dev/sda2              92M  5.7M   81M   7% /boot
/dev/sdd1             230G   31G  188G  15% /home
/dev/sdd2             230G  153G   65G  71% /usr/local

    The free space must exceed 120 MB. Open the Explorer, by using Start → Run... Right-click on the drive where you want to install the program, and choose Properties.

    Figure 1. Properties for a Windows Drive


  2. Physical Memory

    The application needs at lest 1GB of total memory.

    Use Control Panel → System and Maintenance → System to see basic informations about your computer. Check the value reported by Memory (RAM) to be above 1GB.

    Figure 2. System Information


  3. CPU

    The CPU must be at least dualquad core at minimum 1.52GHz.

Installing

To install the program follow these steps.

  1. Download the Linux archive, then run the md5 utility over the downloaded file to check its integrity.

  2. Using a terminal, change directory to /usr/local.

  3. Make a directory photo-album, then extract the downloaded file here, using tar command.

  4. Create a symbolic link from your Desktop to the executable script photo-album.sh

  1. Download the Windows installer executable: photo-album-install.exe

  2. Start the installer, then follow the instructions displayed on screen.
